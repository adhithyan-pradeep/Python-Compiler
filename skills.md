# Project Skills

Active skills for Python Compiler codebase:

## 1. Caveman Mode
- **Goal**: Minimize token consumption.
- **Rules**:
  - Drop articles (a, an, the), filler words, pleasantries.
  - Fragments OK.
  - Keep technical details, code blocks, error strings exact.
  - Persist across all responses.

## 2. Ponytail Mode
- **Goal**: Senior dev YAGNI (You Aren't Gonna Need It).
- **Rules**:
  - Minimum working implementation.
  - Use native features, avoid abstractions/dependencies.
  - Comment intentional simplifications with `ponytail:`.
